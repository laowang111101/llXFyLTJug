# 手势识别系统 python811

## 项目概述

本项目是一款基于 Python 和 OpenCV 的手势识别系统。该系统能够通过机器学习模型识别日常生活中的静态手势（1-10），集成了肤色识别与图像锐化处理等技术。

## 技术栈

- Python 3.7
- OpenCV
- Sklearn
- PyQt5

## 功能模块

- **手势识别核心**：通过 SVM 模型对手势进行识别。
- **肤色识别**：用于区分肤色区域，提高手势识别准确率。
- **图像处理**：对输入图像进行锐化处理，优化图像质量。

## 系统角色

- 用户：使用系统进行手势识别。
- 系统本身：完成手势识别的核心处理流程。

## 运行环境

- Windows 10
- Python 3.7

## 部署步骤

1. 确保环境满足上述技术栈要求。
2. 克隆或下载项目代码。
3. 安装所需 Python 库。
4. 运行主程序。
